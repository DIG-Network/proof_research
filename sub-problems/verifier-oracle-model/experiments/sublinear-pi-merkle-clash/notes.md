# Notes

- **`B(n)`** is only a **strawman** “sublinear-looking” budget; a real **o(n)** proof might still be **`ω(polylog n)`** — the clash is **asymptotic**: **`Θ(n log n)`** dominates **`polylog`** for large **`n`**.
- Does **not** rule out **`O(√n)`** or **`O(n / log n)`** proofs — still **fail** the main **`o(n)`** requirement if constants matter on-chain; check **`main-problem/problem-statement.md`** for exact **`o(n) · λ`** wording.
