# Notes

- **General formula:** For targets **t−1** and **t**, **R = n − |Q|**, **z** ones on **Q**, extensions exist iff **0 ≤ (t−1) − z ≤ R** and **0 ≤ t − z ≤ R**. **Quorum** **impossible** exactly when **t − z > R** i.e. **z < t − R**. Here **t = 6**, **R = 5** ⇒ **z < 1** ⇒ **z = 0**.

- **Link to 036–037:** **037** is **k ≤ n − t** **(4 ≤ 4)** **⇒** **R ≥ t** **⇒** **always** **both** **classes**. **First** **k** **with** **R = t − 1** **(here** **k = 5)** **is** **where** **the** **all-zero** **mask** **(and** **only** **that** **for** **this** **symmetric** **counting)** **breaks** **wt = t** **feasibility**.

- **Adaptive trees:** A **depth-5** **path** **that** **reveals** **five** **zeros** **can** **safely** **REJECT** **quorum**; **any** **path** **with** **≥1** **one** **among** **five** **probes** **still** **hits** **an** **ambiguous** **leaf** **for** **5** **vs** **6** **in** **this** **pure** **coordinate** **model**.
