# Notes

- **Takeaway:** **Any** **single** **modular** **residue** **of** **the** **raw** **count** **k** **(** **with** **modulus** **m ≤ n** **)** **cannot** **implement** **the** **strict** **cut** **k ≥ t** **vs** **k < t** **when** **k** **is** **only** **known** **to** **lie** **in** **[0, n]** **—** **not** **specific** **to** **majority** **t** **or** **n = 10.**
- **Interface:** **Mirrors** **023** **/** **081** **but** **strips** **the** **arithmetic** **to** **“** **every** **gap** **1..n** **crosses** **the** **threshold** **line** **.”**
- **Next:** **If** **a** **protocol** **only** **reveals** **k mod m** **with** **m = O(1),** **pair** **it** **with** **other** **probes** **(** **oracle** **tree** **line** **)** **or** **accept** **Θ(log n)** **bits** **for** **m > n.**
