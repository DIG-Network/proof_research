# Notes — nonadaptive-four-query-participation-indistinguishable

- **Why** **four** **queries:** **n** **−** **|Q|** **≥** **t** **⇔** **|Q|** **≤** **n** **−** **t** **=** **4** **for** **n=10,** **t=6** **—** **exactly** **when** **the** **outside** **has** **room** **for** **t** **ones** **while** **Q** **is** **all** **zero,** **so** **the** **all-zero** **prefix** **on** **Q** **lies** **in** **both** **“below** **threshold”** **and** **“at** **threshold”** **layers** **via** **outside** **weight** **only.**
- **Adaptive** **queries** **not** **modeled** **here** **—** **a** **decision** **tree** **of** **depth** **4** **might** **still** **separate** **some** **pairs** **(different** **game).**
- **Session** **balance:** **follows** **034–035** **(anonymous-quorum-binding** **aggregates)** **with** **a** **verifier-oracle** **query-budget** **template** **without** **new** **spectral** **or** **polynomial** **machinery.**
