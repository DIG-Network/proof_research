# Notes

- **Purpose:** After several heavy **`exists_tree`** runs (**090–093**), this experiment was a lightweight regression that pins the **general combinatorial lemma** behind **`min_d = 1`** when a single **n**-bit XOR (total parity) is allowed (**092**). No new decision-tree search.

- **Not novel cryptography:** The observation is “Hamming weight mod 2 equals global XOR on **{0,1}ⁿ**.” It matters here only as **documented infrastructure** tying **092** to a reusable statement for the verifier-oracle digest.

- **Limits:** Exhaustive checks stop at **n = 18** (combinatorial blow-up). Larger **n** rely on the parity-of-integers tautology; a fully formal proof is one line from **|x| mod 2 = ⊕ᵢ xᵢ**.

- **Next:** Prefer **anonymous-quorum-binding** follow-ups (**078** / **064** injective triple-power sums on **5∪6** shells, or other non-redundant lines in that digest) over more **F₂** tree depth scans unless a new gate class is hypothesized.
