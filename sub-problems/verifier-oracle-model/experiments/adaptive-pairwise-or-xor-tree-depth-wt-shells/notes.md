# Notes

- **Parity check:** For `(6,4)`, shells are **wt 3** (odd) and **wt 4** (even). Global parity is a **linear** functional with **odd-weight** coefficient vector; it is **not** a single pair XOR, but **adaptive** **pair** **XORs** **can** **still** **refine** **until** **leaves** **are** **pure** **—** **min** **depth** **3** **witnesses** **that.**

- **Moral for verifier-oracle-model:** “Non-coordinate” is **not** **monotone** **in** **power:** **AND** **/ ** **OR** **/ ** **XOR** **encode** **different** **invariances** **(complement** **vs** **none).** **Any** **real** **verifier** **resource** **taxonomy** **should** **tag** **which** **equivariance** **class** **a** **probe** **lies** **in.**

- **Relation to 021 (F₂ linear pools):** XOR pair queries **are** **linear** **F₂** **measurements** **with** **even-weight** **masks;** **they** **cannot** **distinguish** **x** **from** **x⊕1,** **which** **is** **exactly** **the** **obstruction** **when** **that** **flip** **exchanges** **the** **two** **shells** **(n=2t−1).**

- **Breakthrough log:** the **n=2t−1** **XOR** **impossibility** **is** **a** **small** **clean** **constraint** **—** **logged** **in** **`BREAKTHROUGHS.md`** **with** **low** **novelty** **confidence** **(standard** **parity** **/ ** **symmetry** **reasoning).**
