# Notes

- **Re-querying** **the** **same** **coordinate** **is** **idle;** **optimal** **trees** **query** **≤** **n** **distinct** **indices** **along** **any** **path.**

- **This** **is** **not** **an** **impossibility** **for** **threshold** **verification** **with** **richer** **oracles** **(parities,** **hashes,** **algebraic** **features)** **—** **only** **the** **coordinate** **probe** **subclass.**

- **Unsticking** **prompt** **“encoding** **change”:** **if** **the** **verifier** **only** **gets** **O(1)** **or** **o(n)** **coordinate** **values** **from** **π,** **the** **hidden** **participation** **vector** **can** **still** **hide** **the** **wt** **5** **vs** **6** **distinction** **(045** **+** **037** **family).**
